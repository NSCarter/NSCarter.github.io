ANDROID APP

In my spare time, I have started to develop an Android app called ‘Find My Item’. The app will be used to help users find where they previously put items by allowing them to take a photo or write a description of where the item is.

This app is my first attempt at using Java and Android Studio and is still a work in progress but I hope to put it on the app store soon. The user will be able to add new or edit existing items which will be stored in a database. An item can then be selected from a recycler view when the user needs to find that item.
